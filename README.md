Install dependencies
```
npm install
```

Update the payload values:
```
const NUMBER_OF_RECORDS = 50000;

const USER_ID = "6a4c27e4-652b-4131-8154-e5bfd6209894";

const KEY_SCHEMA_ID = 3;
const KEY_SCHEMA_VALUE = 7;
```

Run the app
```
node index.js
```
